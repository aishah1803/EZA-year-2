# EZA-year-2

## Year 2 project based on creating a hand hygiene classifier


This project is simple it is a UV‑based hand hygiene assessment system that uses computer vision and simple machine‑learning methods to analyse how well hands have been washed. Using images captured before and after applying fluorescent gel, the pipeline detects and segments the hand, applies thresholding to identify remaining glowing areas, aggregates contamination by hand region, and then outputs both an overall cleanliness rating and clear visual/text feedback to help users improve their hand‑washing technique.
